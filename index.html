콘솔창 사칙연산 계산기

브라우저 개발자 도구의 콘솔에서 start()를 입력하면 계산식을 입력할 수 있습니다.

바로 계산하려면 start("10 + 2 * 3 - 4 / 2")처럼 입력하세요.

Windows

  1. F12 키를 눌러 개발자 도구를 엽니다.
  2. Console 탭을 선택합니다.
  3. start()를 입력하여 계산기를 실행합니다.

Mac

  1. Option + Command + I 키를 눌러 개발자 도구를 엽니다.
  2. Console 탭을 선택합니다.
  3. start()를 입력하여 계산기를 실행합니다.
